# 💻Fuzzy Logic Applications

Bu repository, Fuzzy Logic (Bulanık Mantık) dersi kapsamında yapılan uygulamaları ve projeleri içermektedir. Çalışmalar, bulanık sistemlerin temel adımlarını ve ileri düzey yaklaşımları kapsamaktadır.

## ⌨️ İçerik

Aşağıdaki konular doğrultusunda geliştirilmiş uygulamalarım yer almaktadır:

* **Bulanıklaştırma (Fuzzification)** – Giriş değerlerinin bulanık kümelere dönüştürülmesi
* **Kural Tablosu (Rule Base)** – If-Then kurallarının oluşturulması
* **Bulanık Çıkarım (Fuzzy Inference)** – Kurallar üzerinden sonuç çıkarımı
* **Berraklaştırma (Defuzzification)** – Bulanık sonuçların net değere çevrilmesi
* **Yapay Sinir Ağları (Artificial Neural Networks)**
* **Sinirsel Bulanık Mantık (Neuro-Fuzzy Systems)**

## 👨🏻‍💻 Kullanılan Teknolojiler

Projelerde kullanılan araçlar:

* Python
* NumPy / SciPy
* scikit-fuzzy
* MATLAB

## ℹ️ Amaç

Bu çalışmaların amacı:

* Bulanık mantık sistemlerinin temel yapılarını anlamak
* Teorik bilgiyi uygulamaya dökmek
* Gerçek dünya problemlerine esnek çözümler geliştirmek
